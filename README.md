# Dbaba Site

A polished static marketing site for a logistics and construction company.

## Services

- Logistics
- Plant hire
- Building materials supply
- House construction

## Usage

Open `index.html` in a browser to view the site.
The contact form posts to FormSubmit and shows in-page success or error feedback.

## Files

- `index.html` contains the page structure and content.
- `src/styles.css` contains the visual design and responsive layout.
